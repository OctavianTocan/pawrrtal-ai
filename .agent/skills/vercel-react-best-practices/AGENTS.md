# Vercel React Best Practices

This skill uses progressive disclosure. The complete rule set lives in
`rules/*.md`; read only the rule files relevant to the React or Next.js work
being performed.

Start with `SKILL.md` for the category map and quick reference.

## Rule Index

- `rules/async-*.md` — waterfalls, parallel fetching, Suspense boundaries.
- `rules/bundle-*.md` — import shape, dynamic loading, preload strategy.
- `rules/server-*.md` — server actions, RSC serialization, caching, `after()`.
- `rules/client-*.md` — client data fetching and browser event/storage patterns.
- `rules/rerender-*.md` — render stability, memoization, state and effect shape.
- `rules/rendering-*.md` — rendering performance, hydration, visibility, SVGs.
- `rules/js-*.md` — hot-path JavaScript micro-optimizations.
- `rules/advanced-*.md` — event refs, once-only initialization, latest-value refs.
